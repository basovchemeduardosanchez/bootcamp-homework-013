## How to Install Locally

1. Execute the following files in MySQL Workbench
    ```
    db/schema.sql
    db/seeds.sql
    ```
2. Install the required Node.js modules by executing:
    ```sh
    npm i
    ```
3. Run the server using:
    ```sh
    node server.js
    ```
